Get Started with Lethal Habit: Basic Controls and User Interface Tutorial

Welcome to Lethal Habit, a Metroidvania sidescroller that is still in beta. As such, not everything inside the game is finished yet, but we hope you'll enjoy what's available so far. In this tutorial, we'll go over the basic controls and user interface to get you started.

Basic Controls:

  • A: move left
  • D: move right
  • Space: jump/swim up
  • W (hold): look up
  • S (hold): look down
  • B: attack

User Interface:

The game is separated into different layers, which can be displayed by pressing their corresponding number:

  • 0: real game
  • 1: main menu
  • 2: map
  • 3: skill tree

Skill Tree:

To navigate the skill tree, simply hold left click on a skill to upgrade it. Each upgrade requires one skill tree point, which you can see in the top right corner of the screen. You currently have 100 free skill tree points from the beginning to be able to test everything. Some skills have sub-skills, which can be accessed by right-clicking on the skill. To go back to the top layer, press escape.

We hope this tutorial has been helpful in getting you started with Lethal Habit. Have fun exploring the world of rock music and danger!
